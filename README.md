Проект: прогнозирование биологической активности против вируса гриппа на основе химических дескрипторов.

Выполнены:

разведочный анализ и предобработка данных;
регрессионное моделирование для IC50, CC50 и SI;
классификация по четырём сценариям:
IC50 > медианы;
CC50 > медианы;
SI > медианы;
SI > 8;
сравнение и оценка нескольких алгоритмов машинного обучения.

Результат:
Для каждой задачи определены лучшие модели, проведено сравнение метрик качества и сформулированы выводы о применимости методов для прогнозирования активности и селективности соединений.
